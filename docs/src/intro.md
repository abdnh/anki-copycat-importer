# Introduction

This Anki add-on allows you to import decks from copycat apps such as [AlgoApp](https://www.ankiapp.com/) (formerly AnkiApp) and [Noji](https://noji.io/) (formerly AnkiPro).

## Usage

Visit the page relevant to your app:

- [AlgoApp](algoapp.md)
- [Noji](noji.md)
